## üß© Architecture Overview

**Workflow:**  
Event JSON ‚Üí Validation (schema, timestamp, amount) ‚Üí Canonicalization (sorted keys) ‚Üí SHA-256 hashing (with salt) ‚Üí Memo transaction ‚Üí Immutable Solana ledger proof  

**Tech Stack:**  
- Node.js (v18)  
- @solana/web3.js  
- SHA-256 (crypto module)  
- Solana Devnet  

**Security:**  
- Zero PII exposure (hash-only storage)  
- Local salt generation (non-reversible)  
- All wallet keys encrypted and Devnet-only
